# 1. Project Overview
GRAND BLUE is a Django-based web platform designed to showcase and manage water-based activities in Mauritius. The system includes features for visitors, customers, admins, and a superadmin, providing a complete booking and management solution.

The project was developed using:

1. Django (backend & framework)
2. HTML, CSS, JavaScript, Tailwind CSS (frontend)
3. Django ORM, class-based and function-based views
4. Django authentication
5. ReportLab (PDF generation in admin report)

# 2. Features 
**For Visitors**

Browse and filter activities
View reviews and activity details
Contact support
Use chatbot for queries
Sign up to become a customer

**For Customers**

Login/logout
Book activities
Make payments
Manage profile (edit info, change password)
View and cancel bookings
Leave reviews
Receive notifications

**For Admin**

Manage activities (add, edit, delete)
Manage bookings, payments, customers, reviews
Generate monthly PDF reports

**For SuperAdmin**

Full admin privileges
Access to Django Admin site
Manage booking website


# 3. System Technical Summary
**Models Implemented:** Activity, ActivityImage, ActivityHighlight, Booking, Payment, BookingReview, Notification, User

**Security:** Login required decorators, staff-only access, CSRF protection, password hashing, and Forms

**UX Enhancements:** 
1. Dark mode
2. Responsive Design
3. Search and Filter
4. Dynamic booking updates

# 4. How to run the project
**Requirements:**
1. Python 3.x
2. Django
3. Tailwind
4. ReportLab library installed
5. Google generative AI installed

# 5. Conclusion

GRAND BLUE is a fully functional web application that integrates booking, payment, review, user management, and custom admin tools. It ensures usability, security, and scalability, while addressing both customer and administrative needs.
